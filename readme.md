

Bienvenido a este proyecto — Spa & Beauty.

Este proyecto se basa en intentar asemejarse lo más posible a la referencia dada.

Se ha practicado tanto HTML, CSS, SASS como el desarrollo responsive.
Este último se ha realizado a partir de la resolución Laptop L (1440×824) hacia las demás resoluciones.

Se ha optado por esta metodología en lugar de un desarrollo orientado principalmente a móvil, debido a las exigencias del proyecto para reproducir fielmente la referencia.
